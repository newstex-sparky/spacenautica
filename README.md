# Spacenautica

A Subnautica-style survival game in space with 2D and 3D modes.

## Project Structure

This is a multi-stage development project:
- M0: 2D MVP (completed) — basic survival mechanics
- M1-M5: 3D conversion — various 3D features and improvements

## Development Issues

All issues after #29 are for the 3D conversion following the M1-M5 milestones.

## Quick Start

```bash
cd spacenautica
npm install
npm run dev    # Development server
npm run build  # Production build
```

See PLAN.md for detailed development roadmap.
